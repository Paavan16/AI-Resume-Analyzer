# AI Resume Analyzer & ATS Checker

An AI-powered resume analyzer built using Flask, Python, and local LLMs with Ollama + Llama3.

## Features

- Resume Upload
- ATS Score Calculation
- AI-Powered Resume Suggestions
- Skill Gap Analysis
- Local LLM Integration
- Flask Backend
- Responsive Frontend

## Technologies Used

- Python
- Flask
- Ollama
- Llama3
- HTML/CSS
- PDF Parsing
- ATS Keyword Matching

## Run Locally

```bash
pip install -r requirements.txt
python app.py
